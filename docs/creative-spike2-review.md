# User needs 

## Code/design review Creative Spike week 2

Deze week heb je gewerkt aan de tweede creative coding spike met CSS features niet ouder dan 6 maanden. 

### Aanpak  

Vandaag presenteren jullie de nieuwe creative coding spike aan elkaar op een groot scherm. Elke student heeft ongeveer 5 minuten de tijd en na afloop van alle presentaties kies je individueel twee studenten uit om te reviewen. 

### Presenteren van je werk  

#### Voorbereiding van de presentatie  

Open een browser en maak daar drie tabjes aan: 
- Eentje voor het eindresultaat
- Een andere voor jouw inspiratie (moodboard, figma, schets..)
- En tenslotte open je je code in GitHub van de creative coding spike
